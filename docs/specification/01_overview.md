# OAK BARGAIN 公式HP　プロジェクト概要書

**バージョン：** v1.0
**技術スタック：** Next.js × TypeScript × Three.js / Vercel

---

## 1. プロジェクト概要

本プロジェクトは、ヴィンテージジュエリーブランド「OAK BARGAIN」の公式ウェブサイト（以下、HP）を新規構築するプロジェクトである。

ブランドコンセプト「**時を旅した輝きに、次の物語を**」を体験型のインタラクティブなWebサイトとして表現し、来訪者がブランドの世界観に没入できる体験を提供する。

HPは販売ツールではなく**ブランド体験ツール**として機能させ、実店舗・ポップアップへの誘導を促進することを主目的とする。

---

## 2. ブランド情報

| 項目 | 内容 |
|------|------|
| ブランド名 | OAK BARGAIN |
| ブランドコンセプト | 「時を旅した輝きに、次の物語を」 |
| カテゴリ | ヴィンテージジュエリー |
| 販売形態 | 店頭販売・ポップアップのみ（オンライン販売なし） |
| ターゲット | ヴィンテージジュエリーに関心のある20〜50代女性 |
| 特徴 | 時を経たジュエリーに新しい価値と物語を与えるブランド |

---

## 3. 4ライン構成

| ライン名 | 木の部位 | 価格帯 | コンセプト |
|----------|----------|--------|------------|
| Luxury Line | 花（Flower） | 最上位 | K18・プラチナ等の素材価値を持つ最高峰ライン |
| Premium Line | 葉（Leaf） | 中上位 | 希少性・デザイン性・特別感を兼ね備えたライン |
| Standard Line | 枝（Branch） | 中位 | 日常使いと存在感のバランスを取った定番ライン |
| Entry Line | 幹（Trunk） | エントリー | 初めてのヴィンテージジュエリーに最適な入門ライン |

---

## 4. プロジェクト目的

1. ブランドの世界観（ヴィンテージ・重厚感・物語性）をデジタル空間で体験させる
2. 4ラインのグレード感・差別化を視覚的かつ直感的に伝える
3. 実店舗・ポップアップへの誘導を促進する
4. 「発見する喜び」「出会いの特別感」をインタラクションで演出する
